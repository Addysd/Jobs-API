## User Model ğŸ‘¤

### Register User
- Validate `name`, `email`, and `password` using Mongoose.  
- Hash the password using `bcryptjs`.  

### Login User 
- Find the user in the database.  
- Compare passwords using `bcryptjs`.  
- If credentials are correct, generate a token.
